PROJECT OVERVIEW
----------------
DeepLens is a multi-agent system using Microsoft Semantic Kernel framework that
helps researchers cut through hype and understand what matters in research.

✓ Identifies fundamental problem, research stage, and industry demand
✓ Evaluates researcher history (trend follower, deep specialist, abstraction upleveler)
✓ Assesses technical trends (obsolescence, hype detection, oversupply)

FOUR SPECIALIZED AGENTS
-----------------------

1. TranslationAgent
   Purpose: Simplify research jargon and buzzwords
   Methods:
   - translate(content) - Translate papers/abstracts
   - explain_buzzword(buzzword) - Explain specific terms
   
2. AnalysisAgent
   Purpose: Identify problems, stages, and demand
   Methods:
   - analyze(content, context) - Comprehensive analysis
   - identify_problem_hierarchy(content) - Problem abstraction levels
   Classifications:
   - Research Stage: Exploration / Scaling / Convergence
   
3. ResearcherEvaluationAgent
   Purpose: Evaluate researcher patterns
   Methods:
   - evaluate_researcher(publications, name) - Pattern analysis
   - compare_researchers(researchers_data) - Comparative analysis
   Classifications:
   - Trend Follower: Jumps between hot topics
   - Deep Specialist: Sustained focus on core problems
   - Abstraction Upleveler: Progressive work on fundamental problems
   
4. TrendAssessmentAgent
   Purpose: Assess trends and predictions
   Methods:
   - assess_trend(topic, context) - Trend analysis
   - compare_trends(trends, criteria) - Multi-trend comparison
   - detect_oversupply(area, papers) - Oversupply detection
   Classifications:
   - Trend Status: Emerging / Hyped / Mature / Declining / Obsolete
   - Problem Type: Hard / Engineering / Solved / Fake
